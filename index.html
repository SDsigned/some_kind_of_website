[ SOME KIND OF WEBSITE ] v 1.0

... czyli ewoluujące web CV / portfolio.

Założeniem projektu jest przedstawienie posiadanych umiejętności z zakresu następujacych dziedzin web developingu ???? - HTML - CSS "graphic designingu ???" z użyciem oprogramowania: - photoshop - indesign - ilustrator Kolejnym aspektem niniejszego projektu jest jego ewolucja. Projekt bedzie rozszerzany o kolejne wersje ( z zachowaniem starszych w celach archiwalnych) wraz z postepujacym wzrostem juz posiadanych umiejetnosci oraz nabyciem nowych ( w przyszłości bardziej rozbudowane uzyxie JS czy PHP i oparcie strony o skromny aitorski CMS w celu zmiany chociaz by niniejszego tekstu. Edycje oraz zlozenie calego projektu do kupy odbylo sie za pomoca (Tu ikony programow) oraz wszechwiedzacego google w mysl zasady kto pyta nie bładzi a i zeby otrzumac wlasciwa odpowiedz nalezy wiedziec o co.pytac. Dodarkowa ciekawostka moze byc to.iz projekt w wersji 1.0 z zalozenia zawiera jak najmniej elementow hraficznych w formach. Mimo i.z w niektorych przypadkach moze wydawac sie iz zastosowanie grafiki ulatwilo by sprawe lub poproatu skrocilo kod o kilka linijek staram sie tego unikac i osiagnac to.przez odpowiednie zastosowanie css i html (patrz ikony zmiany koloru strony). Niektore funkcjonalnosci sa stricte kosmetyczne i nie wplywaja w zaden sposob na funkcjonalnosc strony ale sa gdyz uwazalem iz warto sie ich nauczyc w celu poznania pewnych zasad dzialania (patrz zmiana koloru strony)
